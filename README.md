## 📌 Descripción

Este proyecto es un **Sistema de Punto de Venta (POS)** diseñado para una papelería, desarrollado en **Java**.  
Permite gestionar productos, ventas, clientes y generar reportes básicos para el control del negocio.

Está enfocado en ofrecer una solución simple, rápida y eficiente para pequeños y medianos negocios.

---

## ⚙️ Características

- 🛍 Registro de ventas en tiempo real  
- 📦 Gestión de productos (altas, bajas, stock)  
- 👥 Administración de clientes  
- 🧾 Generación de tickets de venta  
- 📊 Reportes básicos de ventas  
- 💾 Persistencia de datos en base de datos
